Stand-app
=========

Simple Dropwizard standup application with ReactJS and MongoDB


### 1. Build the jar

```
> mvn clean install
```

### 2. Spin-up the app

```
> java -jar stand-app-1.0-SNAPSHOT.jar server configuration.yml
```

### 3. Give it a try

http://localhost:9000/assets/index.html
